Serveur DNS avec BIND9 (Debian 11)

Configuration complète d’un serveur DNS maître sous Linux.

# Étape 1 : Installation de BIND

sudo apt update
sudo apt install bind9

# Étape 2 : Configuration des zones

sudo nano /etc/bind/named.conf.local
zone "beroi-guit.local" {
    type master;
    file "/etc/bind/db.beroi-guit.local";
};

# Fichier de zone db.beroi-guit.local

$TTL    604800
@       IN      SOA     ns.beroi-guit.local. admin.beroi-guit.local. (
              3         ; Serial
         604800         ; Refresh
          86400         ; Retry
        2419200         ; Expire
         604800 )       ; Negative Cache TTL
@       IN      NS      ns.beroi-guit.local.
@       IN      A       192.168.1.10
ns      IN      A       192.168.1.10

# Vérification & redémarrage

sudo named-checkconf
sudo systemctl restart bind9

Test final avec nslookup :

nslookup
> server 192.168.1.10
> ns.beroi-guit.local

✔ Résolution correcte = DNS opérationnel
